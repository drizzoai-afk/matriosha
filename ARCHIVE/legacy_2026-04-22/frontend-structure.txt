# Matriosha Frontend Structure (Next.js 15 + Clerk + Supabase)

/app
  /dashboard
    page.tsx          # Protected dashboard route (shows user status & vault info)
  layout.tsx          # Root layout with ClerkProvider and global styles
  page.tsx            # Landing page with SignIn/SignUp modals
  globals.css         # Tailwind imports and base styles

/components           # Reusable UI components (Shadcn/UI based)
  /ui
    button.tsx
    card.tsx
    input.tsx
    badge.tsx
  StorageTierVisualizer.tsx  # Visual component for memory tiers
  VaultIntegrityCard.tsx     # Component showing Merkle tree status

/lib
  supabase.ts         # Supabase client initialization (SSR-safe)
  utils.ts            # CN utility for Tailwind classes

/middleware.ts        # Next.js Middleware: Auth protection for /dashboard and cookie sync

/public               # Static assets (favicons, og-images)

.env.example          # Template for environment variables (Clerk, Supabase keys)
next.config.ts        # Next.js configuration
tailwind.config.ts    # Tailwind CSS configuration (Dark theme, custom colors)
tsconfig.json         # TypeScript configuration
package.json          # Dependencies: next, react, @clerk/nextjs, @supabase/ssr, lucide-react

```

Questa è la struttura "vanilla" e pulita che abbiamo concordato. È pronta per essere popolata con la logica di business e lo styling definitivo. 🚀
